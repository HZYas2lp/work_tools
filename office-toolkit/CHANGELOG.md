# 项目开发日志

## 版本 1.0.0 - 2026-06-02

### ✨ 新功能

#### Excel 列匹配工具 (核心功能)
- 新增 Excel 列匹配工具，实现类似 Excel VLOOKUP 功能
- 支持两种比对模式：
  - **单文件多工作表模式**：使用同一 Excel 文件的不同工作表进行比对
  - **双文件比对模式**：分别上传两个 Excel 文件进行比对
- 三种匹配方式：
  - **精确匹配**：完全相等时匹配
  - **模糊匹配**：包含关系匹配（主表值包含参照表值或反之）
  - **正则匹配**：支持自定义正则表达式进行匹配
- 功能特点：
  - 支持拖拽上传和点击上传
  - 支持选择多个返回列
  - 表头重复时自动添加"_参照"后缀区分
  - 支持导出结果为 CSV 文件
  - 100% 纯前端处理，数据安全不上传服务器

#### 技术优化
- 引入 xlsx 库替代自定义 Excel 解析，提高解析准确性和兼容性
- 支持多种格式：.xlsx, .xls, .csv

### 📁 更新的文件

- `tools/excel/excel-vlookup.vue` - Vue 组件版本
- `standalone.html` - 独立运行版本（已更新）
- `excel-vlookup-simple.html` - 简化测试版本
- `excel-vlookup-debug.html` - 调试版本（带详细日志）

### 🎯 使用场景

- 跨表数据合并
- 多数据源信息匹配
- Excel 数据整合与处理

### 🛠️ 技术改进

- 使用业界标准的 xlsx 库解析 Excel 文件
- 改进导出格式，使用 UTF-8 BOM 和 \r\n 换行确保 Excel 正常打开
- 优化 DOM 操作，释放 URL 对象
